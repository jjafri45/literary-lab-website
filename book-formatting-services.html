Author Questions

Book Formatting FAQ

What does a book formatter actually do?
A formatter prepares the inside of the book for professional reading and printing, including spacing, margins, chapter starts, page numbers, and export files that meet platform requirements.
Will my file work for Amazon KDP?
We prepare files to current KDP specifications and revise technical file issues when a platform flags a production problem caused by our setup.
Do I need formatting if I already wrote the manuscript in Word?
Usually yes. A manuscript file and a finished print interior are not the same thing. Word is where the text is written, but print production needs deliberate layout decisions and export handling.
